# Credit calculator

Through this calculator, you can obtain the value of the installments for any credit amount you wish to apply for.
following the French amortization system which is widely used by banks around the world for all types of
credits, the current interest rate of the United States Federal Reserve is shown as information.
You can generate as many credits as you want and they will be stored.

## Facility

No installation is needed for this site. The HTML will work in any browser.
